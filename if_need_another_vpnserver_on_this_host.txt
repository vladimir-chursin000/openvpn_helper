1) Create dir='ovpn-server0' (for example) and copy all content of dir='ovpn-server' (exceptions: cli_certs, pki_root) to dir='ovpn-server0'.
2) Configure new 'ovpn-server0' (change conf-file 'vpn.env'). Main params for change: VPN_SERVER_NAME_S, VPN_PORT_S, VPN_NETWORK_S, VPN_DHCP_IP_S,
VPN_INTERNAL_NETWORK_ROUTE_S, VPN_NETWORK_MASQUERADE_SRC_S, VPN_INTERNAL_NETWORK_MASQUERADE_DST_S, VPN_INTERNAL_IP_DEV_NAME_S, VPN_SERVER_INT_FIREWALLD_ZONE_S, VPN_PORT_CL.
3) Run 'full_setup_vpn_server.sh'.
4) Follow instructions from file 'README_MAIN.txt' (generated by 'full_setup_vpn_server.sh').
